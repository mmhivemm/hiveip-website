The Hive IP solution

An integrated, end-to-end unique-coding platform.

Code generation, print (in your factory or via your packaging supplier), cloud validation with fraud monitoring, and a Consumer Support Panel — from one team, all the way through. Two print routes feed the same closed-loop validation backbone, so your choice of where to print doesn’t change anything downstream.

How it fits together

The closed-loop architecture, end to end.

From the moment a unique code is generated to the moment a consumer redeems it — one continuous, secure system.

Whichever print route you choose, validation, fraud monitoring, the API and the Consumer Support Panel are identical.

The four pillars

Each pillar deepens into its own page.

In-factory print (ACG)

The recommended primary route. Our small ACG hardware plugs into your existing BBD printers and generates codes on the fly, alongside the BBD on the outside of the pack.

What is an ACG?

Packaging-supplier print

For low-volume programmes, pilots, line-constraint cases, or brands that prefer inside-pack printing. We supply the codes; the supplier prints them; we still handle everything downstream.

When this fits

Cloud validation & security

Algorithmic validation on AWS, exposed via a single API. Fraud monitoring, optional AI Vision audit workflows, full audit trails, 99.85% SLA. Same regardless of print route.

How validation works

Consumer Support Panel

A web interface for your consumer-support team to check codes, view consumer history, and issue replacement codes — all audited.
